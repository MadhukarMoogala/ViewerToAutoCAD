Design Automation — One-Time Setup

Run this page once to deploy the TransformPoint appbundle and register the ImportMarkups and PlotToPDF activities in your APS account. Users never need to visit this page.

Step 1 — Build the plugin

In the repo, run:
dotnet build TransformPoint/TransformPoint.csproj -c Release
The bundle zip will be at:
TransformPoint/bin/Release/net8.0-windows/TransformPoint.bundle.zip

Step 2 — Upload bundle & register activities

Select the TransformPoint.bundle.zip produced by the build above. This deploys it to APS Design Automation and registers both activities.

⚠ Cleanup — Delete ALL DA resources

Calls DELETE /forgeapps/me — wipes every bundle, activity, and alias registered under this APS app. Use only during development to start fresh. You must run Setup again afterward.
